# Stockwise

## English Version

### Project Description
Stockwise – is a desktop application for managing and consolidating material requirements across multiple Excel documents. Designed for production planning and supply chain management, it helps optimize inventory control and production calculations.

### Features
- **Multi-Document Integration**: Combine data from multiple Excel files with automatic duplicate detection
- **Smart Material Merging**: Automatic summation of quantities for identical nomenclature items
- **Production Scaling**: Calculate material requirements for different production volumes (10, 100, 1000 units, etc.)
- **Advanced Search**: Quick search through all materials and components
- **Export Capabilities**: Export consolidated data to Excel and Word formats
- **Real-time Calculations**: Instant updates when modifying production quantities

### Technical Stack
- **Backend**: Python 3.10
- **GUI Framework**: PyQt5
- **Data Processing**: Pandas
- **Architecture**: MVC (Model-View-Controller)

## Русская версия

### Описание проекта
Stockwise – это настольное приложение для управления и консолидации требований к материалам по нескольким документам Excel. Разработанный для планирования производства и управления цепочками поставок, он помогает оптимизировать контроль запасов и расчеты производства.

### Features
- **Многодокументированная интеграция**: объединить данные из нескольких файлов Excel с автоматическим дублированным обнаружением
- **Слияние интеллектуального материала**: автоматическое суммирование количеств для идентичных номенклатурных элементов
- **Производственное масштабирование**: Рассчитайте требования к материалам для различных объемов производства (10, 100, 1000 штук и т.д.)
- **Расширенный поиск**: Быстрый поиск по всем материалам и компонентам
- **Экспортные возможности**: Экспорт консолидированных данных в форматы Excel и Word
- **Расчеты в реальном времени**: мгновенные обновления при изменении производственных величин

### Technical Stack
- **Backend**: Python 3.10
- **GUI Framework**: PyQt5
- **Data Processing**: Pandas
- **Architecture**: MVC (Model-View-Controller)
